The first time you use this program:
  * Install Python: https://www.python.org/
  * To support the use of colors: right-click in this folder, click "Open in Terminal," and enter <pip install -r requirements.txt>. If you don't do this, the program will still run, but it will look less pretty.

To run the program, double-click run_trait_picker.cmd.

If you want to add or remove character traits, edit exquisite_character.txt and save. Each trait is given one line.
